# 类数组，arguments

类数组：

类数组必须有几个组成部分：

# 1，属性要为索引属性。（就是数字属性）
# 2，一定要有 length 属性。
# 3，最好再加上 push 方法。

类数组的特性：

# 1，可以利用属性名模拟数组的特性。
# 2，可以动态增长 length 属性。

类数组应用：

# 1，arguments
# 2，DOM 方法，所能生成所有类似于数组的对象，都是类数组

类数组趣事：

如果一个类数组有了 push 方法后，在加上一个 splice 方法，那么这个对象在外形上就和数组一模一样了。

实参列表：arguments

在每一个函数里面，都有一个系统自动创建好的类数组：arguments。

function fn(a, b) {
  // arguments --> [1, 2, 3]  实参列表
  // fn.length --> 2          代表形参的长度
}

arguments 和 形参，是存在映射关系的。（一个变，另一个也跟着变，但是他们不是一个东西，类似于照镜子）

function fn(a, b) {
  console.log(arguments); // [1, 2]
  a = 3;
  console.log(arguments); // [3, 2]
  arguments[1] = 4;
  console.log(arguments); // [3, 4]
}
fn(1, 2);

需要注意的是：

实参列表有几个， arguments 在系统创建的时候就有几个。

只有当 arguments 和 形参，在数量上是相等的时候。才会存在映射关系。

如果不相等，前面的映射，后面的不映射。

function fn(a, b) {
  console.log(arguments); // [1]
  b = 2;
  console.log(arguments[1]); // undefined
  console.log(arguments); // [1]
}
fn(1);
