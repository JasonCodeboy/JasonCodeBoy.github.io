# Vue.extend 用法

我们可以看到，component 只是一个对象，但是我们可以通过 Vue.extend() 来扩展。CompVue 是 Vue 这个类的一个子类。

在对象 component 里面，已经预先设置了一些属性。

因为如果我们直接 new Vue()，其实它是有一些内置的配置，并没有上面我们申明的 props，也没有 data。

我们通过继承 Vue，得到了一个 CompVue。

通过 Vue.extend()，我们可以做到一些属性合并。

我们还可以有种用法： 我们可以在一个组件的内部通过 extends，去指定它从哪个组件进行继承，也一样可以做到属性合并。

我们主要什么时候去使用 extend 呢？

我们开发了一个组件，这个组件功能其实已经比较完善，但是这个组件有时是一个比较公用的组件。

每个地方都可以把它加进去，功能比较广泛，使用起来也有非常多的配置项。

如果这个时候，这个组件在你的业务里面开发不需要用到这么多配置项，而且很多你都可以给它设置默认值，而且你也可能需要去拓展一些它的属性。

那么你就可以通过先去 extend 这个组件，然后你在去把你想要覆盖的一些东西给它覆盖掉。

通过这种方式，你就可以达到这个组件的一个拓展功能，而不需要你再去重新写一个新的组件。

这就是 extend 的功能。
