2. 作用域插槽

<!-- index.vue -->
<template>
  <div>
    <ScopedSlotDemo :url="website.url">
      <template v-slot="slotProps">{{ slotProps.slotData.title }}</template>
    </ScopedSlotDemo>
  </div>
</template>

<script>
import ScopedSlotDemo from "./ScopedSlotDemo";
export default {
  components: {
    ScopedSlotDemo,
  },
  data() {
    return {
      name: "jason",
      website: {
        url: "http://www.jason100.cc",
        title: "jason",
        subTitle: "jason 个人博客",
      },
    };
  },
};
</script>

<!-- ScopedSlotDemo.vue -->
<template>
  <a :href="url">
    <slot :slotData="website">{{ website.subTitle }}</slot>
  </a>
</template>

<script>
export default {
  props: ["url"],
  data() {
    return {
      website: {
        url: "http://jason100.cc",
        title: "Jason100",
        subTitle: "轻量级富文本编辑器",
      },
    };
  },
};
</script>

  首先在 ScopedSlotDemo 组件的 slot 里面定义一个动态的属性，然后给要传入的参数。属性名称可以自己取。然后在 index 组件内的 ScopedSlotDemo 定义一个 template，在 template 属性上定义一个 v-slot，属性名称也可以自己取，然后用 slotProps.slotData.title。slotData 需要和子组件里面的对应起来，要不然找不到。子组件的 slotData 对应的是整个 website，然后我们就可以通过 slotData.title 获取整个作用域的数据
