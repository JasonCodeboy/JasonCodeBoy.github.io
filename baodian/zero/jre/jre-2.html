# 渲染页面的过程

# 渲染过程 - 1

1. 根据 html 代码生成 DOM Tree

2. 根据 CSS 代码生成 CSSOM

3. 将 DOM Tree 和 CSSOM 整合形成 Render Tree

# 渲染过程 - 2

1. 根据 Render Tree 渲染页面

2. 遇到 <script> 则暂停渲染，优先加载并执行 JS 代码，完成后再继续

3. 直至把 Render Tree 渲染完成

# 为何建议把 CSS 放在 head 中？

把 css 规则和 css 代码在 dom 树生成完成之前就加载完毕，当 dom 树生成完以后直接和所有的 css 整合生成一个 Render Tree
